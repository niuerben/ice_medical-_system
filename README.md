# 药品销售管理系统 - 小组活动大纲

## 一、项目概述
**项目名称**：基于Java Swing的药品销售管理系统  
**开发模式**：C/S架构（客户端/服务器）  
**技术栈**：Java SE、Swing GUI、面向对象编程、基础网络通信模拟

---

## 二、项目核心思路
1. **模拟真实流程**：从登录→浏览商品→查看详情→购物车→订单提交的完整电商流程
2. **分层架构**：
   - 表示层：Swing界面
   - 业务逻辑层：Java类处理业务
   - 数据模拟层：集合/文件模拟数据持久化
3. **模块化开发**：各功能模块独立开发，最后整合

---

## 三、分步实施计划

### 第一阶段：项目准备与环境搭建（1-2天）
**目标**：建立项目基础结构
- [ ] 创建标准的Java项目结构
- [ ] 学习MVC模式在Swing中的应用
- [ ] 掌握Git基础协作操作
- [ ] 确定包结构：
  - `com.medicine.client` 客户端界面
  - `com.medicine.server` 服务模拟
  - `com.medicine.entity` 实体类
  - `com.medicine.util` 工具类

**需掌握技术**：
- Java项目结构规范

---

### 第二阶段：数据模型与实体类设计（2-3天）
**目标**：建立系统的数据基础
- [ ] 设计核心实体类：
  - `User`（用户：id, 用户名, 密码, 角色）
  - `Medicine`（药品：id, 名称, 类别, 价格, 库存, 描述）
  - `CartItem`（购物车项：药品, 数量）
  - `Order`（订单：订单号, 用户, 药品列表, 总价, 状态）
- [ ] 设计数据模拟存储：
  - 使用`HashMap`/`ArrayList`模拟数据库
  - 实现数据初始化工具类

**需掌握技术**：
- Java面向对象编程
- 集合框架（List, Map）
- 封装、构造函数、Getter/Setter
- 静态数据初始化
